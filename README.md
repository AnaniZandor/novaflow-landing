NovaFlow – Landing Page de Gestion de Projets
Bienvenue sur le dépôt officiel de **NovaFlow**, une landing page moderne et responsive conçue pour présenter une solution de gestion de projets. 
Ce projet a été réalisé dans le cadre de mon stage en **Full Stack Development** chez **DecodeLabs**.
Le site est accessible en ligne via GitHub Pages :https://ananizandor.github.io/novaflow-landing/
Fonctionnalités principales
-  Design **responsive** (mobile-first)
-  Mode **sombre / clair** avec mémorisation en `localStorage`
-  Menu de navigation **hamburger** pour mobiles
-  Animations CSS fluides (hover, apparition)
-  Accessibilité améliorée (attributs `aria`, focus visible)
-  Sections claires : Héros, Fonctionnalités, Étapes, Témoignages, Appel à l’action
-  Pied de page avec liens sociaux et mentions légales
