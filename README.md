API RESTful pour la gestion des utilisateurs - Projet 2 DecodeLabs.
Cette API a été développée dans le cadre de mon stage **Full Stack Development** chez **DecodeLabs**. Elle permet de gérer des utilisateurs via des opérations CRUD complètes.


Technologies utilisées

 **Node.js** =>Environnement d'exécution JavaScript 
 **Express** => Framework web pour la création d'API 
 **CORS** => Gestion des requêtes cross-origin 
 **Dotenv** => Gestion des variables d'environnement

 Prérequis

- Node.js (v14 ou supérieur)
- npm (v6 ou supérieur)

---

## Installation et démarrage

### 1. Cloner le dépôt

```bash
git clone https://github.com/AnaniZandor/novaflow-landing.git
cd novaflow-landing
git checkout backend-api 

npm install

créé un fichier .env
